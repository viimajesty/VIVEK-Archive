As you all know, our school is making the shift to Microsoft Teams from Google Meet. Get it here!

Microsoft Teams

Download Microsoft Teams from VIVEK Archive



Downloads for Windows 10/11 and MacOS 12/11/10


           

Installation Instructions for Windows



On a Windows computer, double-click the installer file, Teams_windows_x64(.exe) and Microsoft Teams should automatically install for you.

The file should be in your "Downloads" folder.



Installation Instructions for Mac



On a Mac, double-click on the installer file, Teams_osx(.pkg) and follow the on-screen instructions to complete the installation.

The file should be in your "Downloads" folder.
