Important: Microsoft Teams for Windows only works with Windows 10 and 11. Other OSes are not guaranteed to be supported. Windows 7 has reached EOL and all users running Windows 7 are advised to update to Windows 10. If you have a valid Windows 7 Lincense, you can update for free.

Installation Instructions
Windows users can double-click the Teams_windows_x64(.exe) file which is located in their downloads folder and MS Teams will install automatically.

MacOS users, double-click the Teams_osx(.pkg) file which will be located in your downloads folder. Then follow the on-screen instructions.

Linux users will have to use the command-line. Type the following Code:
wget https://packages.microsoft.com/repos/ms-teams/pool/main/t/teams/teams_1.3.00.25560_amd64.deb
Once done, type this and press enter:
sudo dpkg -i teams_1.3.00.25560_amd64.deb
Please visit Microsoft's official Linux installation page
